# 🌱 Agro Forte 2026 — Futuro Sustentável

## 📖 Sobre o Projeto

O projeto **"Agro Forte 2026 — Futuro Sustentável"** aborda a importância da inovação tecnológica, da sustentabilidade e do agronegócio moderno para o desenvolvimento da sociedade.

O site apresenta como o uso de tecnologia no campo, aliado a práticas sustentáveis, pode aumentar a produtividade agrícola ao mesmo tempo em que preserva os recursos naturais.

Durante o desenvolvimento, foram utilizados conhecimentos de **HTML, CSS, JavaScript e Inteligência Artificial Generativa**, que auxiliaram na criação da estrutura inicial do site, posteriormente adaptada, personalizada e expandida conforme as ideias do aluno.

---

## 🎯 Objetivos

- Desenvolver competências em programação web.
- Aplicar conceitos de HTML, CSS e JavaScript na prática.
- Utilizar Inteligência Artificial como ferramenta de apoio ao desenvolvimento.
- Promover reflexões sobre sustentabilidade e inovação no agronegócio.
- Implementar recursos de acessibilidade e experiência do usuário (UX).
- Exercitar criatividade, organização e personalização de código.

---

## 💻 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Inteligência Artificial Generativa

## 🚀 Funcionalidades do Projeto

- Página única (Landing Page).
- Layout responsivo para dispositivos móveis e computadores.
- Sistema de navegação com seções organizadas.
- Sistema de acessibilidade (quando aplicado no projeto):
  - Ajuste de fonte.
  - Modo de alto contraste.
- Carrossel interativo de conteúdos.
- Sistema de FAQ (Perguntas Frequentes).
- Animações suaves de interação.
- Conteúdo dinâmico com JavaScript.
- 🎮 Quiz interativo sobre agro sustentável.
- 🤠 IA Fazendeiro (chat interativo com respostas por palavras-chave).

- **index.html** → Estrutura e conteúdo da página.
- **style.css** → Estilização, layout e responsividade.
- **script.js** → Funcionalidades, interatividade e lógica do site.
